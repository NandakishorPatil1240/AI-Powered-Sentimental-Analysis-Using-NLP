# 🤖 AI Powered Sentiment Analysis Using NLP

An AI-powered Sentiment Analysis application that predicts whether a given text expresses a **Positive**, **Negative**, or **Neutral** sentiment using Natural Language Processing (NLP) and Machine Learning.


## 📌 Project Overview

This project analyzes textual data such as customer reviews, social media posts, product feedback, and comments to determine the underlying sentiment.

The system performs text preprocessing, feature extraction using TF-IDF, and sentiment prediction using a trained Machine Learning model.


## 🚀 Features

- Text Cleaning,
- Tokenization,
- Stopword Removal,
- Lemmatization,
- TF-IDF Vectorization,
- Machine Learning Classification,
- Real-time Sentiment Prediction,
- Streamlit Web Application,
- Easy-to-use Interface

## 📂 Dataset

The dataset contains text reviews and sentiment labels.


## ⚙️ Workflow

1. Data Collection,
2. Data Cleaning,
3. Text Preprocessing,
4. TF-IDF Feature Extraction,
5. Model Training,
6. Model Evaluation,
7. Save Model,
8. Deploy using Streamlit


## 🧠 Machine Learning Algorithms
